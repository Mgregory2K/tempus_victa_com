# Tempus Victa // Cognitive OS

**Operational Status: v3.3.2 (Synthesis Link)**

Tempus Victa is a local-first cognitive operating system. We have now reached the **Synthesis Link** phase, transforming the Bridge into a two-way neural interface for immediate life ingestion.

## 🧠 Twin+: The Digital Counterpart
Twin+ is the soul of the system. It is a persistent personal cognitive model that learns and replicates your identity, decision patterns, and dialect. Johnny 5 (J5) serves as the optional conversational alias for this entity.
- **Layered Memory**: Stratifies information into Working, Episodic, and Semantic layers for human-like recall.
- **Neural Merge**: v3.3.2 introduces non-destructive ID-based merging. No data is overwritten; it is only reinforced.
- **Identity Graph**: Stores a vector-based model of user preferences, risk tolerance, and decision-making patterns.
- **Predictive Optimization**: Future iterations will enable Twin+ to automatically schedule tasks, estimate durations, and resolve calendar conflicts based on high-fidelity Map/Intel integration.

## 🛰️ The Bridge: Strategic Cockpit
The primary expression layer of Twin+. It synthesizes data from all cognitive modules:
- **Personalized Greetings**: Recognizes the user and time of day for high-fidelity interaction.
- **Temporal Timeline**: Compact feed of unified Google Calendars with high-density view optimization.
- **Working Memory**: A "Loose Signals" ticker for non-project tasks (Gas, Groceries, etc.). Supports instant manual and voice ingestion.
- **Strategic Procrastination**: Upcoming "Push To..." functionality will allow for deliberate rescheduling of loose thoughts directly from the cockpit.

## 🏗️ Core Modules
- **Ready Room**: High-fidelity deliberation and research module. Supports the **Ready Room Protocol** (Holodeck simulation).
- **Signal Bay**: The triage deck where all inbound life signals (Email, manual inputs) are weighted and routed.
- **Clock Tower**: The temporal container for the **Sovereign Ledger**, your immutable neural record.
- **The Mirror**: The deck where your Identity Graph weights are reflected back to you.
- **Command**: Shared council access for Root and Authorized Identities. Shareable cross-session via root-level server persistence.

## 🎙️ Universal Ingestion (The Record Button)
A floating, visual-frequency monitored record button:
- **"Cork it"**: Routes thoughts to Spatial Memory.
- **"Task"**: Creates tactical objectives.
- **Intelligent Defaulting**: Speaking without a prefix automatically manifests a "Loose Signal" in Working Memory.

## 📡 Sovereign Cloud Sync (Mothership)
Your entire OS state—including the **Sovereign Ledger**, API keys, and Admin lists—is synchronized via a private vault in your Google Drive (AppData). This uses the `drive.appdata` scope to ensure Twin+ follows your identity across any hardware surface without centralized server dependency.
