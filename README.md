# Tempus Victa // Cognitive OS

**Operational Status: v3.0.0 (Command & Control / PWA Strategy)**

Tempus Victa is a local-first cognitive operating system. The system has reached a critical milestone: a high-density Strategic Cockpit (The Bridge) and universal intent ingestion.

## 🚀 The Bridge: Cognitive Command Center
The Bridge is the expression layer of your digital twin (Twin+). It synthesizes data from all cognitive modules:
- **Interactive Metrics**: "Unlocked," "Focus," "Projects," and "Signals" are now clickable, providing deep-link navigation to functional modules.
- **Temporal Timeline**: A live feed of your Google Calendar, summarized for actionable execution.
- **Latest Triumph**: A persistence-driven display of your most recent win to maintain momentum.
- **Synthesized Directive**: Real-time advice from Twin+ based on system-wide signals.

## 🎙️ Universal Ingestion (The Record Button)
A floating, visual-frequency monitored record button allows for nonlinear input:
- **"Cork it"**: Automatically routes thoughts to the Spatial Memory (Corkboard).
- **"Task"**: Automatically creates high-priority objectives.
- **Triage**: All other inputs are routed to the Signal Bay for processing.

## 📱 Mobile & Integration Strategy
Tempus Victa is built as a **Progressive Web App (PWA)**.
- **Cross-Platform**: A single code base serves both desktop and mobile (Android/iOS).
- **Native-Like**: Supports "Add to Home Screen" for a full-screen, native feel on your Android device.
- **Identity Sync**: Uses Google Identity to bridge the gap between your phone and the mothership.

## 📡 Sovereign Cloud Sync
Data persistence is unified across devices via a private vault in your Google Drive (AppData), ensuring your memory follows your identity.

- **Domain**: [https://tempusvicta.com](https://tempusvicta.com)
- **Environment**: Windows Server + Caddy (SSL) + NSSM (Persistence)
- **Production Port**: `3010`
