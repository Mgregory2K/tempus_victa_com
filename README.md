# Tempus Victa // Cognitive OS

**Operational Status: v2.9.8 (Live Intelligence Cockpit)**

Tempus Victa is a local-first cognitive operating system. The **Bridge** has been successfully overhauled into a strategic cockpit, and the **Daily Brief** is now powered by a live intelligence API.

## 🚀 The Bridge: A Cognitive Command Center
The Bridge presents a high-level, synthesized overview of your life's data, including:
- **High-Level Analytics**: Metrics for Cognitive Focus and System Efficiency.
- **Neural Entropy Graph**: A real-time visualization of your cognitive load.
- **System Status Ledger**: An at-a-glance table of Active Projects, Pending Signals, and recent Wins.

## 🌤️ Morning Glory: Actionable Daily Brief
A summonable modal that provides a real-time, location-aware intelligence summary:
- **Live Data**: Fetches Weather, Traffic, News, and Events via the `/api/intel` route.
- **Financial Analysis**: Presents a high-risk/high-reward investment table.
- **User-Centric**: Remembers your ZIP code and is designed for future user customization.

## 📡 Sovereign Cloud Sync
Data persistence is unified across devices via a private vault in your Google Drive, accessible via **Push** and **Pull** commands in the main header.

## 🧬 Deployment Specs (Windows Azure VM)

- **Domain**: [https://tempusvicta.com](https://tempusvicta.com)
- **Environment**: Windows Server
- **Reverse Proxy**: Caddy (Automated SSL)
- **Process Manager**: NSSM (Windows Service)
