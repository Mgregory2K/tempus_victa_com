# UDL v2 — One-ZIP Production Drop

## One thing to do (Windows)
Double-click:
- `START_UDL_WINDOWS.bat`

It will:
- create a local `.venv` (first run only)
- install requirements
- start the server on port 9100
- open the UI in your browser

URL:
- http://127.0.0.1:9110/ui/shell/index.html
- On LAN: http://<your-ip>:9110/ui/shell/index.html

## Notes
- Default theme is LIGHT (Azure/Jira/ServiceNow admin feel)
- Dark theme still exists via `data-theme="dark"` on `<html>` if you ever want it.
