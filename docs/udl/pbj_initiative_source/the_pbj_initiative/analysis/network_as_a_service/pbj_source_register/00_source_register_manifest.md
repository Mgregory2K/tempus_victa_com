# Source Register Manifest — Network as a Service (NaaS)

## Rule 0 — Epistemic Order
Evidence comes first. PBJ may weight evidence, but PBJ may not invent it.

## Evidence sufficiency gate
- PASS: Tier A/B sources cover all scored axes, including contrarian evidence.
- WARN: Some axes rely on Tier C or incomplete contrarian evidence; verdict allowed with confidence penalty.
- FAIL: Major axes lack sources; verdict NOT allowed.

## Tier guidance
- Tier A: peer‑reviewed research, official standards, government publications, primary datasets
- Tier B: reputable industry research, major market reports, well‑documented vendor definitions, credible surveys
- Tier C: blogs/opinion/practitioner guidance (supporting, not primary)

## Required metadata per source
- Source ID
- Title
- Author/Organization
- Publication date (if available)
- Link
- Accessed date
- Tier
- PBJ axis tags
- Key findings
- Limits/caveats
