# PBJ Post-Decision Actions

## If GO
- Artifacts to create
- Build phase entry

## If NO-GO
- Archive rationale
- Revisit conditions

## If HOLD
- Explicit triggers to re-evaluate

## Audit-Grade Learning Loop (PBJ v2.0)
- Update the Source Register with:
  - What was correct / incorrect
  - Which sources misled (if any) and why
  - New disconfirming evidence discovered
  - Outcome metrics that should become future evidence
- If a verdict changed, record:
  - Trigger
  - Delta in sources
  - Delta in assumptions
  - Delta in scores/confidence
