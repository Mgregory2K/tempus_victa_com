# Source Register Manifest (PBJ v2.0)

## Rule 0 — Epistemic Order
**Evidence comes first.** PBJ may weight evidence, but may not invent it.

## Required outputs
- Claims Matrix (every scored claim maps to at least one source)
- Bibliography (full citations + access dates)
- Evidence Sufficiency Result (PASS/WARN/FAIL)
- Assumptions Register (explicit items PBJ is guessing)

## Source classes (minimum set)
1. Behavioral / Cognitive (how humans behave)
2. Market / Economic (what survives financially)
3. Governance Precedent (where governance works)
4. Regulatory / Legal (constraints and liabilities)
5. Contrarian / Disconfirming (best counter-evidence)

## Quality tiers (guidance)
- Tier A: peer-reviewed research; official standards; government publications; primary datasets; audited financials
- Tier B: reputable journalism; major think tanks; university reports; high-quality industry reports
- Tier C: blogs, opinion pieces, anecdotal reports (allowed only to frame hypotheses)

## Evidence sufficiency gate
- PASS: Tier A/B sources cover all scored axes; disconfirming evidence included
- WARN: some axes rely on Tier C or assumptions; verdict allowed with explicit confidence penalty
- FAIL: major axes lack sources; verdict NOT allowed (Exploratory/Hold only)

## Required metadata per source
- Title
- Author/Organization
- Publication date
- URL or location
- Access date
- Tier (A/B/C)
- Tags (axis mapping)
- Notes / key findings
